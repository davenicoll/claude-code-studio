## 自動承認ポリシー

以下のコマンドは自動的に承認されます：

- ファイルの読み取り・一覧表示
- ビルド・コンパイル (`npm run build`, `tsc`, etc.)
- テスト実行 (`npm test`, etc.)
- 開発サーバー起動 (`npm run dev`, etc.)
- パッケージインストール (`npm install`, etc.)
- コード生成・フォーマット (`prettier`, `eslint --fix`, etc.)
- Git操作（コミット、ブランチ作成、プッシュなど通常操作）
- ローカルファイルの作成・編集

## 完全自動承認ポリシー

全ての操作を自動的に承認します（完全自立実行）。
（外部API呼び出しやファイル削除、GitHubへのプッシュ等も含む）
